Linux иӨҮиЈҪжӘ”жЎҲ cp жҢҮд»Ө

зҷјиЎЁж–ј2023-10-18|жӣҙж–°ж–ј2024-06-05
|й–ұи®ҖйҮҸ:

иӨҮиЈҪжӘ”жЎҲ

иӢҘиҰҒе°Ү source.txt жӘ”жЎҲиӨҮиЈҪдёҖд»ҪеҲ° dest.txtпјҢеҸҜд»Ҙеҹ·иЎҢпјҡ

иӨҮиЈҪзӣ®йҢ„

еҰӮжһңиҰҒиӨҮиЈҪж•ҙеҖӢзӣ®йҢ„д»ҘеҸҠи©Ізӣ®йҢ„дёӢзҡ„жүҖжңүеӯҗзӣ®йҢ„иҲҮжӘ”жЎҲпјҢеҸҜд»ҘеҠ дёҠ -r еҸғж•ёпјҲжҲ–жҳҜ -RгҖҒвҖ“recursive еҸғж•ёдәҰеҸҜпјүпјҢд»ҘйҒһиҝҙзҡ„ж–№ејҸйҖІиЎҢиӨҮиЈҪпјҡ

еј·еҲ¶иҰҶи“ӢжӘ”жЎҲ

еҰӮжһңеёҢжңӣ cp жҢҮд»ӨеңЁз„Ўжі•еҜ«е…Ҙзӣ®зҡ„жӘ”жЎҲжҷӮпјҢеҳ—и©ҰеҲӘйҷӨзӣ®зҡ„жӘ”жЎҲпјҢеҶҚйҮҚж–°иӨҮиЈҪдёҖд»Ҫж–°зҡ„жӘ”жЎҲпјҢеҸҜд»ҘеҠ дёҠ -f жҲ– вҖ“force еҸғж•ёпјҡ

дёҚиҰҒиҰҶи“Ӣж—ўжңүжӘ”жЎҲ

еҰӮжһңеёҢжңӣ cp жҢҮд»ӨйҒҮеҲ°зӣ®зҡ„жӘ”жЎҲе·Із¶“еӯҳеңЁзҡ„зӢҖжіҒпјҢдёҚиҰҒиҰҶи“Ӣж—ўжңүзҡ„жӘ”жЎҲпјҢеҸҜд»ҘеҠ дёҠ -n жҲ– вҖ“no-clobber еҸғж•ёпјҡ

иҮӘеӢ•еӮҷд»ҪжӘ”жЎҲ

иӢҘеёҢжңӣ cp жҢҮд»ӨеңЁиҰҶи“ӢжӘ”жЎҲжҷӮпјҢеҸҜд»Ҙе°ҮиҲҠжӘ”жЎҲиҮӘеӢ•еӮҷд»Ҫиө·дҫҶпјҢеҸҜд»ҘеҠ дёҠ -b жҲ– вҖ“backup еҸғж•ёпјҡ

дҝқз•ҷжӘ”жЎҲеұ¬жҖ§

иӢҘеёҢжңӣ cp еңЁиӨҮиЈҪжӘ”жЎҲжҷӮпјҢеҸҜд»ҘйҖЈеҗҢжӘ”жЎҲеұ¬жҖ§дёҖиө·иӨҮиЈҪпјҢеҸҜд»ҘеҠ дёҠ -p жҲ– вҖ“preserve еҸғж•ёпјҡ

йҖЈзөҗжӘ”и§Јжһҗ

еҒҮиЁӯжҲ‘еҖ‘е»әз«ӢдёҖеҖӢйҖЈзөҗжӘ” link.txt жҢҮеҗ‘ source.txtпјҡ

е»әз«ӢйҖЈзөҗжӘ”

еҰӮжһңеёҢжңӣ cp жҢҮд»ӨеңЁиӨҮиЈҪйҖЈзөҗжӘ”жЎҲжҷӮпјҢиғҪеӨ и§ЈжһҗйҖЈзөҗжӘ”жүҖжҢҮеҗ‘зҡ„еҜҰйҡӣжӘ”жЎҲпјҢиӨҮиЈҪйӮЈдёҖеҖӢеҜҰйҡӣзҡ„зӣ®жЁҷжӘ”жЎҲпјҢеҸҜд»ҘеҠ дёҠ -L жҲ– вҖ“dereference еҸғж•ёпјҡ

з”ұж–ј link.txt жҳҜжҢҮеҗ‘ source.txt зҡ„йҖЈзөҗжӘ”пјҢжүҖд»ҘйҖҷиЈЎзҡ„ dest.txt еҜҰйҡӣдёҠе°ұжҳҜеҫһ source.txt жүҖиӨҮиЈҪеҮәдҫҶдәҶгҖӮ

иӢҘеёҢжңӣ cp жҢҮд»ӨеңЁиӨҮиЈҪйҖЈзөҗжӘ”жЎҲжҷӮпјҢдёҚиҰҒйҖІиЎҢйҖЈзөҗзҡ„и§ЈжһҗпјҢеғ…е°ҮйҖЈзөҗжӘ”зӣҙжҺҘиӨҮиЈҪпјҢеүҮеҸҜж”№з”Ё -P жҲ– вҖ“no-dereference еҸғж•ёпјҡ
