Flutter еӯёзҝ’зҙҖйҢ„ - Day 2 еҹәзӨҺжҰӮеҝө

Flutterзҡ„жһ¶ж§Ӣ

Flutter жҳҜдёҖеҖӢи·Ёе№іеҸ° UI е·Ҙе…·зө„пјҢиЁӯиЁҲз”Ёж–је…ҒиЁұеңЁдҪңжҘӯзі»зөұпјҲдҫӢеҰӮ iOS е’Ң Androidпјүд№Ӣй–“йҮҚиӨҮдҪҝз”ЁзЁӢејҸзўјпјҢеҗҢжҷӮд№ҹе…ҒиЁұжҮүз”ЁзЁӢејҸзӣҙжҺҘиҲҮеә•еұӨе№іеҸ°жңҚеӢҷд»ӢжҺҘгҖӮзӣ®жЁҷжҳҜи®“й–Ӣзҷјдәәе“ЎиғҪеӨ жҸҗдҫӣеңЁдёҚеҗҢе№іеҸ°дёҠж„ҹиҰәиҮӘз„¶зҡ„й«ҳж•ҲиғҪжҮүз”ЁзЁӢејҸпјҢеңЁе®ғеҖ‘еӯҳеңЁе·®з•°зҡ„ең°ж–№жҺЎз”Ёе·®з•°пјҢеҗҢжҷӮзӣЎеҸҜиғҪе…ұз”ЁзЁӢејҸзўјгҖӮ

еңЁй–Ӣзҷјжңҹй–“пјҢFlutter жҮүз”ЁзЁӢејҸжңғеңЁ VM дёӯеҹ·иЎҢпјҢи©І VM жҸҗдҫӣзӢҖж…ӢзҶұйҮҚијүи®ҠжӣҙпјҢиҖҢдёҚйңҖиҰҒе®Ңе…ЁйҮҚж–°з·ЁиӯҜгҖӮе°Қж–јзҷјиЎҢпјҢFlutter жҮүз”ЁзЁӢејҸжңғзӣҙжҺҘз·ЁиӯҜжҲҗж©ҹеҷЁзўјпјҢз„Ўи«–жҳҜ Intel x64 жҲ– ARM жҢҮд»ӨпјҢйӮ„жҳҜйҮқе°Қз¶Іи·Ҝзҡ„ JavaScriptгҖӮжӯӨжһ¶ж§ӢжҳҜй–Ӣж”ҫеҺҹе§Ӣзўјзҡ„пјҢжҺЎз”ЁеҜ¬й¬Ҷзҡ„ BSD жҺҲж¬ҠпјҢдёҰж“Ғжңүи“¬еӢғзҷјеұ•зҡ„з¬¬дёүж–№еҘ—д»¶з”ҹж…Ӣзі»зөұпјҢиЈңе……ж ёеҝғзЁӢејҸеә«еҠҹиғҪгҖӮ

жӯӨжҰӮи§ҖеҲҶзӮәеӨҡеҖӢеҚҖж®ө

  • ең–еұӨжЁЎеһӢпјҡе»әж§Ӣ Flutter зҡ„йғЁеҲҶгҖӮ
  • еҸҚжҮүејҸдҪҝз”ЁиҖ…д»ӢйқўпјҡFlutter дҪҝз”ЁиҖ…д»Ӣйқўй–Ӣзҷјзҡ„ж ёеҝғжҰӮеҝөгҖӮ
  • е°Ҹе·Ҙе…·з°Ўд»ӢпјҡFlutter дҪҝз”ЁиҖ…д»Ӣйқўзҡ„еҹәжң¬е»әж§ӢеҚҖеЎҠгҖӮ
  • жёІжҹ“иҷ•зҗҶпјҡFlutter еҰӮдҪ•е°Ү UI зЁӢејҸзўјиҪүжҸӣжҲҗеғҸзҙ гҖӮ
  • е№іеҸ°еөҢе…ҘеҷЁжҰӮи§Җпјҡи®“иЎҢеӢ•е’ҢжЎҢдёҠеһӢдҪңжҘӯзі»зөұеҹ·иЎҢ Flutter жҮүз”ЁзЁӢејҸзҡ„зЁӢејҸзўјгҖӮ
  • е°Ү Flutter иҲҮе…¶д»–зЁӢејҸзўјж•ҙеҗҲпјҡй—ңж–ј Flutter жҮүз”ЁзЁӢејҸеҸҜз”Ёзҡ„дёҚеҗҢжҠҖиЎ“зҡ„иіҮиЁҠгҖӮ
  • ж”ҜжҸҙз¶Іи·Ҝпјҡй—ңж–јзҖҸиҰҪеҷЁз’°еўғдёӯ Flutter зү№жҖ§зҡ„зөҗи«–жҖ§иӘӘжҳҺгҖӮ

жһ¶ж§Ӣең–еұӨ

Flutter иў«иЁӯиЁҲзӮәеҸҜж“ҙе……зҡ„ең–еұӨзі»зөұгҖӮе®ғд»ҘдёҖзі»еҲ—зҚЁз«Ӣзҡ„зЁӢејҸеә«еӯҳеңЁпјҢжҜҸеҖӢзЁӢејҸеә«йғҪдҫқиіҙж–јеә•еұӨең–еұӨгҖӮжІ’жңүд»»дҪ•ең–еұӨе…·жңүе°ҚдёӢж–№ең–еұӨзҡ„зү№ж¬ҠеӯҳеҸ–ж¬ҠпјҢиҖҢдё”жһ¶ж§ӢеұӨзҡ„жҜҸеҖӢйғЁеҲҶйғҪиЁӯиЁҲзӮәеҸҜйҒёе’ҢеҸҜжӣҝжҸӣзҡ„гҖӮ

https://flutter.dev.org.tw/resources/architectural-overview

Flutterзҡ„жһ¶ж§ӢеҲҶжҲҗдёүеұӨпјҡ

Framework

Framework еұӨжҳҜ Flutter зҡ„й«ҳеұӨйғЁеҲҶпјҢдё»иҰҒжҳҜз”ұ Dart иӘһиЁҖж’°еҜ«гҖӮйҖҷдёҖеұӨжҸҗдҫӣдәҶж§Ӣе»әжҮүз”ЁзЁӢејҸзҡ„еҹәжң¬зөҗж§Ӣе’Ңе·Ҙе…·гҖӮе®ғеҢ…еҗ«дәҶд»ҘдёӢе№ҫеҖӢеӯҗеұӨпјҡ

  • WidgetsпјҡFlutter жҸҗдҫӣдәҶиұҗеҜҢзҡ„ Widget йӣҶеҗҲпјҢз”Ёж–јж§Ӣе»әиӨҮйӣңзҡ„ UI е…ғд»¶гҖӮйҖҷдәӣ Widget жҳҜ Flutter жҮүз”ЁзЁӢејҸзҡ„ж ёеҝғпјҢжүҖжңүзҡ„иҰ–ең–е’Ңдә’еӢ•йғҪз”ұйҖҷдәӣ Widget зө„жҲҗгҖӮ

  • RenderingпјҡйҖҷеҖӢеӯҗеұӨиІ иІ¬е°Ү Widget жёІжҹ“еҲ°иһўе№•дёҠгҖӮе®ғиҷ•зҗҶдҪҲеұҖгҖҒз№ӘиЈҪе’ҢеӢ•з•«зӯүж“ҚдҪңпјҢзўәдҝқ Widget иғҪеӨ жӯЈзўәйЎҜзӨәе’Ңдә’еӢ•гҖӮ

  • Animation and GesturesпјҡжҸҗдҫӣдәҶеӢ•з•«е’ҢжүӢеӢўзҡ„ж”ҜжҸҙпјҢи®“й–ӢзҷјиҖ…иғҪеӨ иј•й¬Ҷең°ж·»еҠ иұҗеҜҢзҡ„еӢ•з•«ж•Ҳжһңе’Ңиҷ•зҗҶз”ЁжҲ¶зҡ„жүӢеӢўијёе…ҘгҖӮ

Engine

Engine еұӨжҳҜ Flutter зҡ„ж ёеҝғпјҢе®ғз”ұ C++ з·ЁеҜ«пјҢдёҰдё”иІ иІ¬дҪҺеұӨж¬Ўзҡ„жёІжҹ“е’Ңијёе…Ҙиҷ•зҗҶгҖӮйҖҷдёҖеұӨжҸҗдҫӣдәҶи·Ёе№іеҸ°зҡ„жёІжҹ“е’Ңеҹ·иЎҢз’°еўғпјҢдҪҝеҫ— Flutter жҮүз”ЁиғҪеӨ еңЁдёҚеҗҢзҡ„е№іеҸ°дёҠйҒӢиЎҢгҖӮEngine еҢ…еҗ«д»ҘдёӢдё»иҰҒзө„д»¶пјҡ

  • SkiaпјҡдёҖеҖӢй–Ӣжәҗзҡ„ 2D ең–еҪўеә«пјҢз”Ёж–јз№ӘиЈҪжүҖжңүзҡ„ең–еҪўе…ғзҙ гҖӮSkia жҸҗдҫӣдәҶй«ҳж•Ҳзҡ„з№Әең–жҖ§иғҪпјҢзўәдҝқ Flutter жҮүз”ЁиғҪеӨ жөҒжҡўйҒӢиЎҢгҖӮ

  • Dart RuntimeпјҡйҒӢиЎҢ Dart зЁӢејҸзўјзҡ„еҹ·иЎҢз’°еўғпјҢиІ иІ¬ Dart зЁӢејҸзўјзҡ„з·ЁиӯҜе’Ңеҹ·иЎҢгҖӮ

  • TextпјҡжҸҗдҫӣдәҶж–Үеӯ—жёІжҹ“зҡ„ж”ҜжҸҙпјҢеҢ…жӢ¬иӨҮйӣңзҡ„ж–Үеӯ—дҪҲеұҖе’ҢжёІжҹ“гҖӮ

Embedder

Embedder еұӨиІ иІ¬е°Ү Flutter жҮүз”ЁеөҢе…ҘеҲ°зү№е®ҡзҡ„ж“ҚдҪңзі»зөұдёӯгҖӮйҖҷдёҖеұӨжҸҗдҫӣдәҶиҲҮеә•еұӨе№іеҸ°дә’еӢ•зҡ„д»ӢйқўпјҢдҪҝеҫ— Flutter еҸҜд»ҘеңЁдёҚеҗҢзҡ„ж“ҚдҪңзі»зөұдёҠйҒӢиЎҢгҖӮEmbedder еҢ…еҗ«д»ҘдёӢеҠҹиғҪпјҡ

  • Platform Channelsпјҡз”Ёж–ј Dart е’ҢеҺҹз”ҹе№іеҸ°д»Јзўјд№Ӣй–“зҡ„йҖҡдҝЎгҖӮйҖҷдҪҝеҫ— Flutter жҮүз”ЁиғҪеӨ иӘҝз”ЁеҺҹз”ҹе№іеҸ°зҡ„ APIпјҢдёҰдё”и®“еҺҹз”ҹе№іеҸ°иғҪеӨ иӘҝз”Ё Dart зЁӢејҸзўјгҖӮ

  • Event Loopпјҡиҷ•зҗҶж“ҚдҪңзі»зөұзҡ„дәӢд»¶пјҢеҰӮијёе…ҘдәӢд»¶е’Ңз•«йқўжӣҙж–°пјҢдёҰе°ҮйҖҷдәӣдәӢд»¶еӮійҒһзөҰ Flutter жҮүз”ЁгҖӮ

  • Window Managementпјҡз®ЎзҗҶжҮүз”Ёзҡ„зӘ—еҸЈе’ҢиҰ–зӘ—пјҢдҪҝеҫ— Flutter жҮүз”ЁиғҪеӨ еңЁдёҚеҗҢе№іеҸ°дёҠд»ҘжӯЈзўәзҡ„еӨ§е°Ҹе’ҢдҪҚзҪ®йЎҜзӨәгҖӮ

и·Ёе№іеҸ°й–Ӣзҷје·Ҙе…·жҜ”ијғ

иҲҮ APP й–Ӣзҷјзӣёй—ңзҡ„дё»иҰҒи·Ёе№іеҸ°й–ӢзҷјжЎҶжһ¶е’Ңе·Ҙе…·еҢ…жӢ¬д»ҘдёӢе№ҫеҖӢпјҡ

1. Flutter

https://docs.flutter.dev/

  • иӘһиЁҖпјҡDart
  • жҸҸиҝ°пјҡз”ұ Google й–Ӣзҷјзҡ„жЎҶжһ¶пјҢеҸҜд»Ҙж§Ӣе»әй«ҳж•ҲиғҪзҡ„еҺҹз”ҹжҮүз”ЁгҖӮе…¶зү№й»һеҢ…жӢ¬иұҗеҜҢзҡ„ widget еә«е’Ңе„Әз§Җзҡ„ Hot Reload еҠҹиғҪгҖӮ
  • зү№й»һпјҡ
    • й«ҳж•ҲиғҪпјҢзӣҙжҺҘз·ЁиӯҜжҲҗеҺҹз”ҹд»ЈзўјгҖӮ
    • ж“ҒжңүиұҗеҜҢзҡ„й җиЁӯе…ғд»¶е’ҢжҸ’д»¶гҖӮ
    • ж”ҜжҸҙ iOSгҖҒAndroidгҖҒWeb е’ҢжЎҢйқўжҮүз”ЁгҖӮ

2. React Native

https://reactnative.dev/

  • иӘһиЁҖпјҡJavaScript
  • жҸҸиҝ°пјҡз”ұ Facebook й–ӢзҷјпјҢдҪҝз”Ё JavaScript е’Ң React дҫҶж§Ӣе»әеҺҹз”ҹжҮүз”ЁгҖӮе…·жңүе»ЈеӨ§зҡ„зӨҫзҫӨе’ҢиұҗеҜҢзҡ„з¬¬дёүж–№иіҮжәҗгҖӮ
  • зү№й»һпјҡ
    • дҪҝз”Ё JavaScript е’Ң React дҫҶй–ӢзҷјгҖӮ
    • еҸҜд»Ҙе…ұдә«еӨ§йғЁеҲҶд»Јзўјж–ј iOS е’Ң Android е№іеҸ°гҖӮ
    • йңҖиҰҒйҖҸйҒҺ JavaScript Bridge иҲҮе№іеҸ°жәқйҖҡгҖӮ

3. Ionic

https://ionicframework.com/

  • иӘһиЁҖпјҡJavaScript (Angular, React, Vue)
  • жҸҸиҝ°пјҡеҹәж–ј web жҠҖиЎ“зҡ„жЎҶжһ¶пјҢдҪҝз”Ё HTMLгҖҒCSS е’Ң JavaScript дҫҶж§Ӣе»әи·Ёе№іеҸ°жҮүз”ЁгҖӮдҫқиіҙж–ј Cordova е’Ң CapacitorгҖӮ
  • зү№й»һпјҡ
    • дҪҝз”ЁжөҒиЎҢзҡ„ web жЎҶжһ¶пјҲеҰӮ AngularгҖҒReactгҖҒVueпјүгҖӮ
    • еҸҜд»ҘйҖҡйҒҺ WebView е°ҮжҮүз”ЁеҢ…иЈқжҲҗеҺҹз”ҹжҮүз”ЁгҖӮ
    • ж”ҜжҸҙе»Јжіӣзҡ„жҸ’д»¶е’ҢеҺҹз”ҹеҠҹиғҪиЁӘе•ҸгҖӮ

4. Apache Cordova (PhoneGap)

https://cordova.apache.org/

  • иӘһиЁҖпјҡHTML, CSS, JavaScript
  • жҸҸиҝ°пјҡдҪҝз”Ё web жҠҖиЎ“дҫҶж§Ӣе»әз§»еӢ•жҮүз”ЁпјҢдёҰеҢ…иЈқжҲҗеҺҹз”ҹжҮүз”ЁеңЁеҗ„зЁ®е№іеҸ°дёҠйҒӢиЎҢгҖӮ
  • зү№й»һпјҡ
    • йҖҡйҒҺ WebView жёІжҹ“жҮүз”Ёе…§е®№гҖӮ
    • ж”ҜжҸҙеӨҡзЁ®е№іеҸ°пјҲiOSгҖҒAndroid зӯүпјүгҖӮ
    • жҸ’д»¶зі»зөұиұҗеҜҢпјҢиғҪиЁӘе•Ҹеҗ„зЁ®еҺҹз”ҹеҠҹиғҪгҖӮ

5. Kotlin Multiplatform Mobile (KMM)

https://www.jetbrains.com/kotlin-multiplatform/

  • иӘһиЁҖпјҡKotlin
  • жҸҸиҝ°пјҡз”ұ JetBrains жҸҗдҫӣпјҢе…ҒиЁұдҪҝз”Ё Kotlin дҫҶе…ұдә«жҘӯеӢҷйӮҸијҜд»ЈзўјпјҢеҗҢжҷӮеңЁ Android е’Ң iOS дёҠйҒӢиЎҢгҖӮ
  • зү№й»һпјҡ
    • е°ҲжіЁж–је…ұдә«йӮҸијҜеұӨпјҢUI еұӨдҪҝз”Ёе№іеҸ°еҺҹз”ҹж–№ејҸеҜҰзҸҫгҖӮ
    • жҸҗдҫӣ Kotlin иӘһиЁҖзҡ„жүҖжңүе„ӘеӢўе’ҢеҠҹиғҪгҖӮ
    • иҲҮ Android еҺҹз”ҹй–Ӣзҷјз„Ўзё«ж•ҙеҗҲгҖӮ

6. Unity

https://unity.com/cn

  • иӘһиЁҖпјҡC#
  • жҸҸиҝ°пјҡдё»иҰҒз”Ёж–јйҒҠжҲІй–ӢзҷјпјҢдҪҶд№ҹиғҪз”Ёж–јж§Ӣе»әеҗ„зЁ®и·Ёе№іеҸ°зҡ„жҮүз”ЁгҖӮ
  • зү№й»һпјҡ
    • ж”ҜжҸҙеӨҡзЁ®е№іеҸ°пјҢеҢ…жӢ¬ iOSгҖҒAndroidгҖҒWindowsгҖҒmacOSгҖӮ
    • еј·еӨ§зҡ„ 2D е’Ң 3D жёІжҹ“иғҪеҠӣгҖӮ
    • ж“Ғжңүе»Јжіӣзҡ„жҸ’д»¶е’ҢиіҮжәҗеә«гҖӮ

8. NativeScript

https://nativescript.org/

  • иӘһиЁҖпјҡJavaScript, TypeScript
  • жҸҸиҝ°пјҡе…ҒиЁұдҪҝз”Ё JavaScript жҲ– TypeScript дҫҶж§Ӣе»әеҺҹз”ҹз§»еӢ•жҮүз”ЁгҖӮ
  • зү№й»һпјҡ
    • жҸҗдҫӣе°ҚеҺҹз”ҹ API зҡ„зӣҙжҺҘиЁӘе•ҸгҖӮ
    • еҸҜд»ҘдҪҝз”Ё Angular жҲ– Vue.js дҫҶж§Ӣе»әжҮүз”ЁгҖӮ
    • ж”ҜжҸҙ iOS е’Ң AndroidгҖӮ

Component ж”ҜжҸҙ з”ұ Google й–ӢзҷјпјҢй җиЁӯиұҗеҜҢ йңҖиҰҒз¬¬дёүж–№ж”ҜжҸҙ з”ұ Microsoft ж”ҜжҢҒпјҢиҲҮ .NET з”ҹж…Ӣзі»зөұж•ҙеҗҲ йңҖиҰҒз¬¬дёүж–№ж”ҜжҸҙ йңҖиҰҒз¬¬дёүж–№ж”ҜжҸҙ йңҖиҮӘиЎҢй–Ӣзҷј иұҗеҜҢпјҢйҮқе°ҚйҒҠжҲІй–Ӣзҷј йңҖиҰҒз¬¬дёүж–№ж”ҜжҸҙ
ж–Үд»¶е“ҒиіӘ и©ізҙ°жҳ“и®Җ з¬¬дёүж–№ Library ж–Үд»¶е“ҒиіӘдёҚдёҖ и©ізҙ°жҳ“и®Җ з¬¬дёүж–№ Library ж–Үд»¶е“ҒиіӘдёҚдёҖ з¬¬дёүж–№ Library ж–Үд»¶е“ҒиіӘдёҚдёҖ и©ізҙ°жҳ“и®Җ и©ізҙ°жҳ“и®Җ и©ізҙ°жҳ“и®Җ
зӨҫзҫӨжҲҗзҶҹеәҰ зҷјеұ•дёӯ жҲҗзҶҹ зҷјеұ•дёӯ жҲҗзҶҹ жҲҗзҶҹ зҷјеұ•дёӯ жҲҗзҶҹ зҷјеұ•дёӯ
ж•ҲиғҪ зӣҙжҺҘиҪүжҲҗ native code йңҖйҖҸйҒҺ Bridge иҲҮе№іеҸ°жәқйҖҡ зӣҙжҺҘиЁӘе•ҸеҺҹз”ҹ API еҹәж–ј WebView еҹәж–ј WebView зӣҙжҺҘиЁӘе•ҸеҺҹз”ҹ API й«ҳж•ҲиғҪ 2D е’Ң 3D жёІжҹ“ зӣҙжҺҘиЁӘе•ҸеҺҹз”ҹ API
App еӨ§е°Ҹ ијғе°Ҹ ијғеӨ§ дёӯзӯү дёӯзӯү дёӯзӯү дёӯзӯү ијғеӨ§ дёӯзӯү
зү№иүІиҲҮеӮҷиЁ» й«ҳж•ҲиғҪпјҢиұҗеҜҢзҡ„ widget еә«е’Ң Hot Reload еҠҹиғҪ дҪҝз”Ё JavaScript е’Ң React й–ӢзҷјпјҢе»ЈеӨ§зҡ„зӨҫзҫӨиҲҮиіҮжәҗ з№јжүҝиҮӘ XamarinпјҢиҲҮ Visual Studio з·ҠеҜҶж•ҙеҗҲ дҪҝз”Ё web жҠҖиЎ“ж§Ӣе»әжҮүз”ЁпјҢдҫқиіҙж–ј Cordova е’Ң Capacitor дҪҝз”Ё web жҠҖиЎ“ж§Ӣе»әжҮүз”ЁпјҢе»Јжіӣзҡ„е№іеҸ°ж”ҜжҢҒ е…ұдә«йӮҸијҜеұӨпјҢUI еұӨдҪҝз”Ёе№іеҸ°еҺҹз”ҹж–№ејҸеҜҰзҸҫ йҒҠжҲІй–ӢзҷјзӮәдё»пјҢдҪҶеҸҜж§Ӣе»әеҗ„зЁ®жҮүз”ЁпјҢж”ҜжҢҒеӨҡе№іеҸ° дҪҝз”Ё Angular жҲ– Vue.js ж§Ӣе»әжҮүз”ЁпјҢж”ҜжҸҙ iOS е’Ң Android

зёҪзөҗ

  • Flutter жҳҜдёҖеҖӢеј·еӨ§дё”й«ҳж•Ҳзҡ„жЎҶжһ¶пјҢйҒ©еҗҲж§Ӣе»әй«ҳжҖ§иғҪгҖҒеҺҹз”ҹй«”й©—зҡ„и·Ёе№іеҸ°жҮүз”ЁгҖӮ
  • React Native е…·жңүе»ЈеӨ§зҡ„зӨҫзҫӨе’ҢиіҮжәҗпјҢдҪҝз”Ё JavaScript е’Ң ReactпјҢжҳҜиЁұеӨҡй–ӢзҷјиҖ…зҡ„йҰ–йҒёгҖӮ
  • .NET MAUI з№јжүҝиҮӘ XamarinпјҢиҲҮ .NET е’Ң Visual Studio з·ҠеҜҶзөҗеҗҲпјҢйҒ©еҗҲе·Іжңү C# жҠҖиғҪзҡ„й–ӢзҷјиҖ…гҖӮ
  • Ionic е’Ң Cordova йҒ©еҗҲдҪҝз”Ё web жҠҖиЎ“й–ӢзҷјжҮүз”Ёзҡ„еңҳйҡҠпјҢе…·жңүијғеҘҪзҡ„и·Ёе№іеҸ°ж”ҜжҢҒгҖӮ
  • Kotlin Multiplatform Mobile (KMM) жҸҗдҫӣе…ұдә«жҘӯеӢҷйӮҸијҜзҡ„иғҪеҠӣпјҢйҒ©еҗҲ Android е’Ң iOS еҺҹз”ҹй–ӢзҷјиҖ…гҖӮ
  • Unity йҒ©еҗҲйңҖиҰҒй«ҳж•ҲиғҪ 2D/3D жёІжҹ“зҡ„жҮүз”ЁпјҢе°Өе…¶жҳҜйҒҠжҲІй–ӢзҷјгҖӮ
  • NativeScript е…ҒиЁұдҪҝз”Ё Angular жҲ– Vue.js ж§Ӣе»әжҮүз”ЁпјҢжҸҗдҫӣзӣҙжҺҘиЁӘе•ҸеҺҹз”ҹ API зҡ„иғҪеҠӣгҖӮ

йҖҷдәӣжЎҶжһ¶еҗ„жңүе„ӘеӢўе’ҢйҒ©з”Ёе ҙжҷҜпјҢйҒёж“Үе“ӘдёҖеҖӢеҸ–жұәж–јдҪ зҡ„е…·й«”йңҖжұӮе’ҢжҠҖиЎ“иғҢжҷҜгҖӮеёҢжңӣйҖҷдәӣжҜ”ијғиғҪе°ҚдҪ жңүжүҖе№«еҠ©гҖӮ
