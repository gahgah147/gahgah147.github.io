Nuxt 3 еҜҰжҲ°еӯёзҝ’зӯҶиЁҳ - з’°еўғиЁӯе®ҡ

зҷјиЎЁж–ј2023-12-14|жӣҙж–°ж–ј2024-06-05
|й–ұи®ҖйҮҸ:
Nuxt 32023 iTHome йҗөдәәиіҪVueеүҚз«ҜжЎҶжһ¶

еүҚиЁҖ

йҖҷзҜҮж–Үз« жҳҜжҲ‘еңЁ 2023 iTHome йҗөдәәиіҪзңӢеҲ° Ryan еӨ§еӨ§ еҜ«зҡ„ Nuxt 3 еҜҰжҲ°зӯҶиЁҳ зі»еҲ— ж–Үз« еҒҡзҡ„дёҖеҖӢзҙҖйҢ„пјҢжҲ‘д№ӢеүҚд№ҹйғҪжңүзңӢеҲ°зҷјиЎЁNuxt зӣёй—ңзҡ„ж–Үз« пјҢеҜ«еҫ—йғҪеҫҲз”Ёеҝғд»”зҙ°еҫҲеҖјеҫ—и·ҹи‘—з·ҙзҝ’пјҢжҲ‘иҰәеҫ—жңҖзү№еҲҘзҡ„жҳҜ Nuxt ж–°зҡ„ DevToolsеҠҹиғҪ

Nuxt CLI жҢҮд»Ө

nuxi е…ЁеҗҚзӮә (Nuxt CLI, Nuxt Command Line Interface)пјҢжҳҜз”ұ Nuxt жҸҗдҫӣй–Ӣзҷјзҡ„жЁҷжә–е·Ҙе…·пјҢNuxt CLI е°ұеғҸжҳҜ Vue CLI еҸҜд»Ҙе»әз«Ӣ Vue е°ҲжЎҲпјҢжҲ‘еҖ‘з•¶з„¶д№ҹе°ұеҸҜд»ҘдҪҝз”Ё Nuxt CLI дҫҶе»әз«Ӣ Nuxt зҡ„е°ҲжЎҲгҖӮ

з’°еўғжә–еӮҷ

з’°еўғжә–еӮҷжөҒзЁӢеҰӮдёӢпјҢйҖҡеёёдё»иҰҒжңғз”ЁеҲ°зҡ„nuxi жҢҮд»Өд№ҹжҳҜйҖҷе№ҫеҖӢ

жё…йҷӨ npx еҝ«еҸ–
е»әз«Ӣ Nuxt е°ҲжЎҲ
жӘўжҹҘе°ҲжЎҲзӣёй—ңиіҮиЁҠ
еҚҮзҙҡ Nuxt 3 зүҲжң¬
е•ҹеӢ•й–ӢзҷјдјәжңҚеҷЁ
жё…йҷӨиҮӘеӢ•з”ўз”ҹзҡ„ Nuxt жӘ”жЎҲе’Ңеҝ«еҸ–
з”ўз”ҹйқңж…Ӣз¶Із«ҷдҪҝз”Ёзҡ„й җжёІжҹ“
жү“еҢ…е°ҲжЎҲе»әж§Ӣз”ҹз”ўз’°еўғйңҖиҰҒзҡ„зЁӢејҸеҢ…
й җиҰҪз¶Із«ҷ
й–Ӣзҷје·Ҙе…· Nuxt DevTools

жӣҙеӨҡе®Ңж•ҙзҡ„жҢҮд»ӨиҲҮз”Ёжі•еҸҜд»ҘеҸғиҖғе®ҳж–№ж–Үд»¶гҖӮ

жё…йҷӨ npx еҝ«еҸ– - clear-npx-cache

з•¶дёҖеҲҮжә–еӮҷе°ұз·’еҫҢпјҢдҪ еҸҜд»Ҙеҹ·иЎҢдёӢеҲ—жҢҮд»ӨпјҢе®ғе°Үжңғе®үиЈқжңҖж–°зүҲжң¬зҡ„ Nuxt CLI дёҰеҹ·иЎҢ вҖ“help дҫҶзңӢзӣёй—ңиӘӘжҳҺгҖӮ

еҰӮжһңдҪ жңүзңӢеҲ°еҰӮдёӢең–зҡ„зӣёй—ңжҢҮд»ӨиӘӘжҳҺпјҢе°ұд»ЈиЎЁдҪ е®үиЈқжҲҗеҠҹеӣүпјҒ

дҪҝз”Ё nuxi е»әз«Ӣ Nuxt е°ҲжЎҲиҲҮе•ҹеӢ•й–ӢзҷјдјәжңҚеҷЁ

еҰӮжһңдҪ з¬¬дёҖж¬ЎдҪҝз”Ё Nuxt 3пјҢдҫқж“ҡе®ҳз¶Ізҡ„зҜ„дҫӢпјҢдҪ еҸҜд»ҘеңЁзөӮз«Ҝж©ҹеҹ·иЎҢдёӢеҲ—жҢҮд»ӨпјҢе°ұиғҪй–Ӣе§Ӣе»әз«Ӣ Nuxt зҡ„е°ҲжЎҲпјҡ

е»әз«Ӣзҡ„йҒҺзЁӢ

йҖҷйӮҠйҰ–е…Ҳжңғе•Ҹ Which package manager would you like to use?
йҒёж“Ү npm

Are you interested in participating?

ж“ҚдҪңе®ҢжҲҗд№ӢеҫҢжңғе‘ҠиЁҙдҪ жҺҘдёӢдҫҶеҰӮдҪ•й–Ӣе•ҹе°ҲжЎҲ
вңЁ Nuxt project has been created with the v3 template. Next steps:
вҖә cd nuxt-app
вҖә Start development server with npm run dev

з•¶дҪ еңЁй–ӢзҷјжҷӮпјҢеҸҜд»ҘеңЁе°ҲжЎҲиіҮж–ҷеӨҫе…§еҹ·иЎҢдёӢеҲ—жҢҮд»ӨдҫҶе•ҹеӢ•й–ӢзҷјдјәжңҚеҷЁпјҡ

дёҠиҝ°жҢҮд»ӨпјҢеҜҰйҡӣдёҠжҳҜеҹ·иЎҢ package.json дёӯзҡ„ scripts жүҖе®ҡзҫ©зҡ„ dev е°ҚжҮүи…іжң¬жҢҮд»ӨеҰӮдёӢпјҡ

е…¶еҜҰд№ҹзӯүеҗҢж–јдҪ еңЁе°ҲжЎҲзӣ®йҢ„дёӢеҹ·иЎҢпјҡ

nuxi дёҚеғ…еңЁе•ҹеӢ•й–ӢзҷјдјәжңҚеҷЁиҲҮйғЁзҪІз·ЁиӯҜдёҠйңҖиҰҒдҪҝз”ЁпјҢй–Ӣзҷјзҡ„йҒҺзЁӢдёӯпјҢд№ҹйӮ„жңүиЁұеӨҡдёҚеҗҢзҡ„жҢҮд»ӨиҲҮеҸғж•ёеҸҜд»Ҙж“ҚдҪңдҪҝз”ЁпјҢд»Ҙе№«еҠ©жҲ‘еҖ‘й–Ӣзҷј Nuxt е°ҲжЎҲдёҠзҡ„дҪҝз”ЁгҖӮ

жӘўжҹҘ Nuxt е°ҲжЎҲзӣёй—ңиіҮиЁҠ -info

еҚҮзҙҡ Nuxt 3 зүҲжң¬ -upgrade

йҖҷеҖӢжҢҮд»ӨеҸҜд»Ҙз”ЁдҫҶе°Үзӣ®еүҚе°ҲжЎҲзҡ„ Nuxt 3 еҚҮзҙҡиҮіжңҖж–°зҡ„зүҲжң¬пјҢеҰӮжһңжңүдёҖдәӣеҸҜиғҪиЎҢзӮәиӘҝж•ҙжҲ–дёҚзӣёе®№зҡ„жғ…жіҒпјҢеҸҜд»ҘеҶҚдҫқж“ҡеҜҰйҡӣжғ…еўғжҗӯй…Қ -f, вҖ“force еҸғж•ёдҫҶеј·еҲ¶жӣҙж–°гҖӮ

е•ҹеӢ•й–ӢзҷјдјәжңҚеҷЁ - run dev

дёҖеҖӢж–°е»әз«Ӣзҡ„ Nuxt е°ҲжЎҲпјҢз•¶дҪ иҰҒе•ҹеӢ•й–ӢзҷјдјәжңҚеҷЁпјҢдҪ еҸҜд»ҘдҪҝз”ЁдёӢйқўзҡ„жҢҮд»Өпјҡ

Nuxt зҡ„ Nitro е°ұжңғе№«жҲ‘еҖ‘е•ҹеӢ•й–ӢзҷјдјәжңҚеҷЁпјҢдёҰзӣЈиҒҪ Port: 3000гҖӮ

вҖ“port, -p иЁӯе®ҡзӣЈиҒҪPort

й җиЁӯзӣЈиҒҪ Port: 3000пјҢиӢҘеӣ зӮәиЎқзӘҒжҲ–жңүйңҖиҰҒеҒҡиӘҝж•ҙеҸҜд»ҘдҪҝз”ЁйҖҷеҖӢеҸғж•ё

дҪҝз”Ё npm еҹ·иЎҢ scriptsпјҢеҸҜд»ҘеҠ дёҠе…©еҖӢжёӣиҷҹ вҖ“ д»Ҙж·»еҠ и…іжң¬жҢҮд»ӨеҫҢзҡ„еҸғж•ёгҖӮ

вҖ“host, -h иЁӯе®ҡ дјәжңҚеҷЁдё»ж©ҹеҗҚзЁұ

й җиЁӯдјәжңҚеҷЁдё»ж©ҹеҗҚзЁұзӮә localhostпјҢиӢҘеӣ зӮәдёҖдәӣи·ЁеҹҹжҲ–й©—иӯүзӯүй–ӢзҷјйңҖжұӮпјҢд№ҹеҸҜд»ҘиӘҝж•ҙ

вҖ“open, -o иЁӯе®ҡ й–Ӣе•ҹзҖҸиҰҪеҷЁ

з•¶й–ӢзҷјдјәжңҚеҷЁе•ҹеӢ•жҷӮпјҢй–Ӣе•ҹзҖҸиҰҪеҷЁдёҰе°Һеҗ‘еҲ°й–Ӣзҷјзҡ„з¶ІеқҖгҖӮ

вҖ“https иЁӯе®ҡ е•ҹз”Ёhttps

еңЁй–ӢзҷјжҷӮиӢҘжңүдёҖдәӣй©—иӯүжҲ–и«ӢжұӮйңҖиҰҒ HTTPSпјҢд№ҹеҸҜд»ҘдҪҝз”ЁйҖҷеҖӢеҸғж•ёдҫҶе•ҹз”Ё HTTPSпјҢй җиЁӯжғ…жіҒд№ҹеҗ«жңүиҮӘз°ҪеҗҚзҡ„ SSL иӯүжӣёгҖӮ

жё…йҷӨиҮӘеӢ•з”ўз”ҹзҡ„ Nuxt жӘ”жЎҲе’Ңеҝ«еҸ– -cleanup

Nuxt 3 еңЁе•ҹеӢ•й–ӢзҷјдјәжңҚеҷЁеҫҢпјҢжңғиҮӘеӢ•е»әз«Ӣ .nuxt зӣ®йҢ„иҲҮз”ўз”ҹзӣёй—ңзҡ„жӘ”жЎҲжҲ– TypeScript дҪҝз”Ёзҡ„йЎһеһӢпјҢе»әж§Ӣе°ҲжЎҲжҲ–з”ўз”ҹйқңж…Ӣз¶Із«ҷжҷӮд№ҹжңғе»әз«Ӣ .output жҲ– dist зӣ®йҢ„зӯүпјҢжҲ‘еҖ‘еҸҜд»ҘдҪҝз”ЁжҢҮд»ӨдҫҶжё…йҷӨйҖҷдәӣиҮӘеӢ•з”ўз”ҹзҡ„жӘ”жЎҲе’Ңеҝ«еҸ–гҖӮ

еҲӘйҷӨзҡ„зӣ®йҢ„еҢ…еҗ«еҰӮдёӢпјҡ

  • .nuxt
  • .output
  • dist
  • node_modules/.vite
  • node_modules/.cache

з”ўз”ҹйқңж…Ӣз¶Із«ҷдҪҝз”Ёзҡ„й җжёІжҹ“ -generate

жҲ‘еҖ‘еҸҜд»ҘдҪҝз”Ёд»ҘдёӢжҢҮд»ӨдҫҶй җжёІжҹ“е°ҲжЎҲдёӯзҡ„жҜҸеҖӢи·Ҝз”ұи·Ҝеҫ‘пјҢдёҰе°Үе…¶зөҗжһңе„ІеӯҳеңЁзҙ” HTML з•¶дёӯпјҢе…¶з”ўз”ҹзҡ„жӘ”жЎҲжңғе»әз«ӢеңЁ .output/public иҲҮ dist зӣ®йҢ„пјҢе…©еҖӢзӣ®йҢ„зҡ„жӘ”жЎҲе…§е®№жҳҜдёҖжЁЈзҡ„пјҢдҪ еҸҜд»ҘйҒёж“ҮдҪҝз”Ё .output/public дҪңзӮәйқңж…Ӣз¶Із«ҷзҡ„йғЁзҪІгҖӮ

жү“еҢ…е°ҲжЎҲе»әж§Ӣз”ҹз”ўз’°еўғйңҖиҰҒзҡ„зЁӢејҸеҢ… -build

з•¶е°ҲжЎҲй–ӢзҷјеҲ°дёҖеҖӢйҡҺж®өиҰҒжә–еӮҷйғЁзҪІжҷӮпјҢдҪ жңғйңҖиҰҒжү“еҢ…дёҰе»әж§Ӣз”ҹз”ўз’°еўғжүҖйңҖиҰҒзҡ„зЁӢејҸпјҢдҪ еҸҜд»ҘдҪҝз”Ёд»ҘдёӢжҢҮд»ӨдҫҶз·ЁиӯҜе»әж§ӢгҖӮ

йҖҷеҖӢжҢҮд»Өжңғе»әз«ӢдёҖеҖӢ .output зӣ®йҢ„пјҢе…¶дёӯеҢ…еҗ«жүҖжңүжҮүз”ЁзЁӢејҸгҖҒдјәжңҚеҷЁиҲҮдҫқиіҙй…ҚзҪ®пјҢдҪ еҸҜд»Ҙе°ҮжӯӨзӣ®йҢ„дҪңзӮәз”ҹз”ўз’°еўғдҪҝз”Ёзҡ„зӣ®йҢ„йҖІиЎҢжңҚеӢҷйғЁзҪІгҖӮ

й җиҰҪз¶Із«ҷ -preview

з•¶дҪҝз”Ё nuxi build е°ҲжЎҲжү“еҢ…е»әж§Ӣе®ҢжҲҗеҫҢпјҢдҪ еҸҜд»ҘдҪҝз”ЁдёӢ preview жҢҮд»ӨдҫҶе•ҹеӢ•дјәжңҚеҷЁпјҢй җиҰҪдҪ зҡ„ Nuxt з¶Із«ҷгҖӮ

й–Ӣзҷје·Ҙе…· Nuxt DevTools

йҖҸйҒҺжңҖж–°зүҲжң¬ Nuxt CLI е»әзҪ®зҡ„ Nuxt 3 е°ҲжЎҲе·Із¶“е°Ү Nuxt DevTools й җзҪ®еңЁ Nuxt е°ҲжЎҲдёӯдёҰй җиЁӯзӮәе•ҹз”ЁпјҢеҰӮжһңдҪ зҡ„е°ҲжЎҲзҡ„ Nuxt 3 зүҲжң¬йӮ„жҜ”ијғиҲҠпјҢеҸҜд»ҘжүӢеӢ•е®үиЈқ Nuxt DevToolsпјҢжҲ–дҪҝз”Ёд»ҘдёӢжҢҮд»ӨдҫҶй…ҚзҪ®гҖӮ

й—ңй–ү Nuxt DevTools

еҰӮжһңдҪ жғіиҰҒй—ңй–ү Nuxt DevToolsпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢжҢҮд»ӨдҫҶй—ңй–үпјҢе®ғжңғжҸҗзӨәжӮЁжҳҜеҗҰиҰҒе°Ү ./nuxt.config.ts жӘ”жЎҲе…§зҡ„ devtools.enabled иЁӯе®ҡзӮә false дҫҶй—ңй–ү DevToolsгҖӮ
